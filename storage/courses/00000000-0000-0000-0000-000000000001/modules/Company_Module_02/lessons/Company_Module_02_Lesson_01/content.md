# Урок 1: Алгоритмы машинного обучения

## Введение

В этом уроке мы изучим основные алгоритмы машинного обучения и их применение.

## Линейная регрессия

### Концепция

Линейная регрессия предсказывает непрерывные значения на основе линейной зависимости:

```
y = w₁x₁ + w₂x₂ + ... + wₙxₙ + b
```

где:
- y — предсказываемое значение
- x — признаки
- w — веса
- b — смещение

### Применение

- Предсказание цен
- Прогнозирование продаж
- Анализ трендов

## Логистическая регрессия

### Концепция

Используется для бинарной классификации:

```
P(y=1|x) = 1 / (1 + e^(-z))
```

где z — линейная комбинация признаков.

### Применение

- Классификация спама
- Медицинская диагностика
- Кредитный скоринг

## Деревья решений

### Концепция

Алгоритм, который принимает решения на основе правил:

```
Если признак X > порог:
    Если признак Y > порог:
        Класс A
    Иначе:
        Класс B
Иначе:
    Класс C
```

### Преимущества

- Легко интерпретировать
- Работает с категориальными данными
- Не требует нормализации

## Случайный лес

### Концепция

Ансамбль деревьев решений:

- Создается множество деревьев
- Каждое дерево голосует
- Финальное решение — большинство голосов

### Преимущества

- Высокая точность
- Устойчивость к переобучению
- Работает с большими данными

## Метод опорных векторов (SVM)

### Концепция

Находит оптимальную границу между классами:

- Максимизирует зазор между классами
- Использует ядерные функции для нелинейных данных

### Применение

- Классификация текста
- Распознавание изображений
- Биоинформатика

## K-ближайших соседей (KNN)

### Концепция

Классифицирует объект на основе k ближайших примеров:

1. Найти k ближайших соседей
2. Определить класс большинства
3. Присвоить этот класс объекту

### Применение

- Рекомендательные системы
- Классификация изображений
- Поиск аномалий

## Заключение

Различные алгоритмы машинного обучения подходят для разных задач. Выбор алгоритма зависит от типа данных, размера выборки и требований к интерпретируемости.
