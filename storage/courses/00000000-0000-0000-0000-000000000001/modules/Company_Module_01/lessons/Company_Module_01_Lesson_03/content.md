# Урок 3: Нейронные сети и глубокое обучение

## Введение в нейронные сети

Нейронные сети — это вычислительные модели, вдохновленные биологическими нейронными сетями мозга.

## Архитектура нейронной сети

### Нейрон (узел)

Основной элемент сети:

- **Входы**: Входные данные (x1, x2, ..., xn)
- **Веса**: Параметры модели (w1, w2, ..., wn)
- **Смещение**: Дополнительный параметр (b)
- **Функция активации**: Нелинейная функция (f)
- **Выход**: Результат вычисления

### Слои сети

1. **Входной слой**: Получает исходные данные
2. **Скрытые слои**: Промежуточные вычисления
3. **Выходной слой**: Финальный результат

## Типы нейронных сетей

### Полносвязные сети (Feedforward)

Все нейроны одного слоя связаны со всеми нейронами следующего слоя.

### Сверточные нейронные сети (CNN)

Специализированы для обработки изображений:

- Используют сверточные слои
- Эффективны для распознавания образов

### Рекуррентные нейронные сети (RNN)

Работают с последовательностями:

- Имеют память о предыдущих состояниях
- Используются для обработки текста и речи

### Трансформеры

Современная архитектура для NLP:

- Attention механизм
- Используется в GPT, BERT и других моделях

## Глубокое обучение

Глубокое обучение — это машинное обучение с использованием многослойных нейронных сетей.

### Преимущества

- Автоматическое извлечение признаков
- Высокая точность на сложных задачах
- Способность работать с большими данными

### Применения

- **Компьютерное зрение**: Распознавание объектов, лиц
- **Обработка естественного языка**: Перевод, генерация текста
- **Голосовые помощники**: Siri, Alexa, Google Assistant
- **Автономные системы**: Самоуправляемые автомобили

## Обучение нейронных сетей

### Процесс обучения

1. **Прямой проход (Forward Pass)**: Вычисление выхода сети
2. **Вычисление ошибки**: Сравнение с правильным ответом
3. **Обратный проход (Backpropagation)**: Обновление весов
4. **Повторение**: Итерации до сходимости

### Оптимизаторы

- **SGD**: Стохастический градиентный спуск
- **Adam**: Адаптивный метод оптимизации
- **RMSprop**: Еще один адаптивный метод

## Современные достижения

### Большие языковые модели

- GPT-4, ChatGPT
- Способны генерировать человеческий текст
- Многофункциональные помощники

### Генеративные модели

- Генерация музыки и видео

## Заключение

Нейронные сети и глубокое обучение — это передний край развития ИИ. Понимание этих технологий открывает множество возможностей для инноваций.
