# Урок 2: Машинное обучение - основы

## Что такое машинное обучение?

Машинное обучение (ML) — это подмножество искусственного интеллекта, которое позволяет системам автоматически учиться и улучшаться на основе опыта без явного программирования.

## Типы машинного обучения

### Обучение с учителем (Supervised Learning)

Система обучается на размеченных данных:

- **Классификация**: Определение категории объекта
- **Регрессия**: Предсказание числовых значений

**Примеры:**
- Распознавание спама в email
- Предсказание цены недвижимости

### Обучение без учителя (Unsupervised Learning)

Система находит паттерны в данных без меток:

- **Кластеризация**: Группировка похожих объектов
- **Анализ ассоциаций**: Поиск связей между данными

**Примеры:**
- Сегментация клиентов
- Рекомендательные системы

### Обучение с подкреплением (Reinforcement Learning)

Система учится через взаимодействие с окружением:

- Агент получает награды за правильные действия
- Учится оптимальной стратегии

**Примеры:**
- Игровые боты
- Робототехника

## Основные алгоритмы

### Линейная регрессия

Простой алгоритм для предсказания непрерывных значений:

```
y = ax + b
```

### Деревья решений

Алгоритм, который принимает решения на основе правил:

```
Если условие → то действие
```

### Нейронные сети

Многослойные системы, имитирующие работу мозга

## Процесс машинного обучения

1. **Сбор данных**: Получение релевантных данных
2. **Подготовка данных**: Очистка и предобработка
3. **Выбор модели**: Выбор подходящего алгоритма
4. **Обучение**: Тренировка модели на данных
5. **Оценка**: Проверка качества модели
6. **Развертывание**: Использование модели в продакшене

## Метрики качества

- **Точность (Accuracy)**: Доля правильных предсказаний
- **Точность (Precision)**: Доля истинно положительных среди всех положительных
- **Полнота (Recall)**: Доля найденных положительных случаев
- **F1-мера**: Гармоническое среднее точности и полноты

## Заключение

Машинное обучение — это мощный инструмент для решения сложных задач. Понимание основных концепций ML необходимо для работы с современными ИИ-системами.
